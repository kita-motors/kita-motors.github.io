## プロジェクト概要

「しらせてタケヤくん」は、LINE WORKS API v2.0とAI（Google Gemini）を使用して、チームの日次報告を自動化し、課題を検出して上長をサポートするボットです。

## 主な機能

### AI分析
- **課題検出**: 技術的問題、リソース不足、依存関係、情報不足を自動検出
- **感情分析**: ポジティブ/ネガティブスコアの算出
- **サポート必要度**: 緊急度の自動判定

### 日次報告の自動化
- 朝・昼・夕の定時に報告フォーマットを送信
- 未応答者への段階的リマインダー
- デイリーサマリーを上長に自動通知
- ウィークリーレポートでチーム状況を可視化

### カレンダー連携
- 予定の確認・追加・変更・削除
- 参加者の追加（@メンション対応）
- 自然言語入力（「22日14:00 ミーティング」など）

### 課題管理
- ボタンUIでステータス更新（解決済み/対応中）
- 課題一覧の可視化

### その他
- `/vacation` コマンドで休暇登録・確認
- `/announce` コマンドで一斉連絡（上長用）

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript (strict mode)
- **ホスティング**: Vercel
- **データベース**: Supabase (PostgreSQL)
- **AI**: Google Gemini API
- **メッセージング**: LINE WORKS API v2.0
- **カレンダー**: LINE WORKS Calendar API
- **定期実行**: GitHub Actions (cron)

## タケヤくんシリーズ

- **おしえてタケヤくん**: わからないことを教えてくれる
- **勤サポタケヤくん**: 勤怠管理をサポート
- **しらせてタケヤくん**: 日次報告・課題検出（このプロジェクト）
- **しょうごうタケヤくん**: 経理の突合せ業務
