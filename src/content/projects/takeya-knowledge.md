## プロジェクト概要

「おしえてタケヤくん」は、Next.jsを使用したAIチャットボットアプリケーションです。Gemini APIと連携し、社内ナレッジファイル（就業規則、経費精算、福利厚生など）を参照しながら質問に回答します。

## 主な機能

### AIチャット
- Gemini API (gemini-2.5-flash) との連携
- RAG機能（ナレッジファイル参照）による正確な回答
- システムプロンプトによる回答スタイルのカスタマイズ
- マークダウン形式での回答表示（コードブロック、テーブル対応）

### 回答品質向上
- **ハルシネーション抑制**: Temperature調整（0.1）で一貫性のある回答
- **自己確認ステップ**: 回答前にナレッジファイルとの整合性を確認
- **禁止事項の明示**: 架空の手順やシステム名の生成を防止

### フィードバック機能
- 各回答に「改善が必要」ボタンを表示
- カテゴリ自動推定（変更可能）
- Googleスプレッドシートへのフィードバック保存
- 収集データによるナレッジ・プロンプトの継続的改善

### その他
- クイックリプライ機能
- Docker環境でのデプロイ対応
- リトライロジック（503エラー対応、エクスポネンシャルバックオフ）

## プロジェクト構成

```
takeya-knowledge/
├── frontend/      # チャットUI（Next.js）
├── backend/       # Gemini API連携（Next.js API Routes）
├── knowledge/     # ナレッジファイル（Markdown形式）
└── tools/         # ナレッジ管理・分析ツール
```

## 技術スタック

- **フレームワーク**: Next.js (フロントエンド・バックエンド両方)
- **言語**: TypeScript
- **AI**: Gemini API (@google/generative-ai)
- **フィードバック保存**: Google Sheets API
- **デプロイ**: Vercel / Docker
