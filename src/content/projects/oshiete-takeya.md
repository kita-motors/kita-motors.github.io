## プロジェクト概要

「おしえてタケヤくん」は、社内の問い合わせ対応を効率化するために開発したAIチャットボットです。

## 背景と課題

- 総務・人事への問い合わせが月間500件以上
- 同じような質問への対応で工数が逼迫
- 社内ドキュメントが分散していて情報を探しにくい

## ソリューション

RAG（検索拡張生成）を活用したAIチャットボットを構築し、社内ドキュメントを学習させることで、正確で迅速な回答を実現しました。

## 技術スタック

- **フロントエンド**: Next.js 14, TypeScript, Tailwind CSS
- **バックエンド**: Next.js API Routes
- **AI**: Google Gemini API
- **ベクトルDB**: pgvector (PostgreSQL)
- **認証**: Azure AD (MSAL)
- **インフラ**: Vercel, Supabase

## 成果

- 問い合わせ対応時間を **50%削減**
- 回答精度 **90%以上**
- 月間アクティブユーザー **200名以上**
