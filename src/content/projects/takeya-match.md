## プロジェクト概要

「しょうごうタケヤくん」は、経理部門での日常的なデータ照合作業を効率化するシステムです。2つのCSVファイルをアップロードし、指定したキー列に基づいて自動的にマッチングを行い、差分や不一致を可視化します。

## 主な機能

### 照合タイプ

**銀行データ照合**
- 銀行データと経理データの自動照合
- 日付+金額による一意化処理（Power Query互換）
- 入金・出金の分離処理
- グループマッチング（1対多照合）対応

**買掛金照合**
- 経理データの摘要欄から仕入先名を自動抽出
- 仕入先名+金額による照合
- 金額不一致の検出機能
- 金額のみ一致する候補名の表示

**売掛金照合**
- 経理データの摘要欄から得意先名を自動抽出
- 日付+得意先名+部門によるグループ化
- グループ内の金額合計で照合

### データ表示・操作
- アップロードされたCSVデータをテーブル形式で表示
- 金額の自動フォーマット（カンマ区切り）
- 検索・絞り込み機能（全列対応、リアルタイム検索）
- 日付ソート機能（昇順・降順切り替え）
- 照合結果の色分け表示（一致/不一致/片方のみ）

### エクスポート
- 照合結果のCSVダウンロード（Shift-JISエンコーディングでExcel互換）
- すべての照合結果を一括エクスポート

### ファイル対応
- CSVファイルのドラッグ&ドロップ対応
- Shift-JISエンコーディングの自動検出と変換
- 経理データの特殊フォーマット対応

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS v4
- **CSVパース**: PapaParse
- **文字エンコーディング**: encoding-japanese (Shift-JIS対応)
- **コンテナ**: Docker / Docker Compose
- **デプロイ**: Vercel
