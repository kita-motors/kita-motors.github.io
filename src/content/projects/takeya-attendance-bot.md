## プロジェクト概要

「勤サポタケヤくん」は、ジョブカンの勤怠打刻を自動確認し、LINE WORKS経由で通知・各種申請を行うNext.jsアプリケーションです。

## 主な機能

### 自動監視・通知
- **残業検知と申請促進**: 17:00以降の退勤で未申請の場合に自動通知
- **打刻忘れ警告**: 20:00以降も勤務中ステータスの場合に警告
- **定期実行**: GitHub Actionsで平日17:05-20:05に毎時実行（JST）

### LINE WORKSボタンUI
- **ボタンタップで簡単操作**: 理由選択、確認などをボタンで完結
- **日本語時刻入力**: 「18時半」「17時45分」などスマホで打ちやすい形式に対応

### 残業申請
- 「残業申請」または「残業申請 18:30」でキーワード起動
- 定型文（事務処理、打ち合わせ対応、月次業務など）から理由を選択
- 確認後に自動でジョブカンに申請送信

### 早出申請
- 「早出申請」でキーワード起動
- 開始時刻と理由を入力
- 確認後に自動でジョブカンに申請送信

### 有給申請
- 「有給申請」でキーワード起動
- 有給種類の選択（1日、午前半休、午後半休、0.25日、0.75日、代休）
- 日付入力（「12月26日」「12/26」など複数形式対応）
- 確認後に自動でジョブカンに申請送信

### 複数ユーザー対応
- Vercel KV（Upstash Redis）を使用したユーザー管理
- 各ユーザーごとの個別ジョブカン認証情報管理
- RESTful ユーザー管理API

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **ホスティング**: Vercel
- **データストア**: Vercel KV (Upstash Redis)
- **メッセージング**: LINE WORKS API v2.0
- **勤怠連携**: ジョブカン（Puppeteerによるブラウザ自動操作）
- **定期実行**: GitHub Actions (cron)
